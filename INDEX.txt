   Works without backend! Uses echo responses if API fails

❓ FAQS:

Q: Do I need a backend server to test?
A: No! Open index.html directly. It uses fallback responses.

Q: Does it work on mobile?
A: Yes! iOS Safari & Android Chrome fully supported.

Q: Can I customize it?
A: Yes! CSS colors, JS behavior, HTML text all easily changeable.

Q: How do I connect to OpenClaw?
A: Deploy backend with /api/chat endpoint → Connect to OpenClaw gateway

Q: What if microphone doesn't work?
A: Check browser permissions, ensure HTTPS/localhost, try different browser.

Q: Can I deploy this on my own server?
A: Yes! Copy index.html to any web server, ensure HTTPS.
